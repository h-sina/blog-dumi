# JavaScript
## JS 事件循环机制
一个主栈执行同步任务，一个异步队列执行异步结果。JS是同步的，所以会先执行主栈内全部的同步任务，执行完毕之后才执行异步队列。异步任务又分微任务和宏任务，常见的微任务有promise.then，宏任务有setTimeout、setInternet，遵循执行完一次宏任务就执行完全部的异步队列中的微任务。
